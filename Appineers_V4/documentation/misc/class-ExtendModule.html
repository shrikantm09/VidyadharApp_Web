Class ExtendModule

If all you need to do is add some functionality to an existing module - perhaps add a function or extend the used function. In this case it's better to simply extend the module's class.

Extending a class is nearly identical to replacing a class with a couple exceptions:
     - The class declaration must extend the parent class.
     - Your new class name and filename must be prefixed with Cit_ .

You can extend a module by:
     - Extending Controllers
     - Extending Models
     - Extending Views

How to extend a module using ConfigureIT?
  Any module can be extend by ConfigureIT, follow given instructions:
   - Go to extended classes and add new controller.
   - Select one module. After selection, you will be able to extend controller,model or view of that particular module.
   - You can create new method or can overwite old methods as per your requirements.
